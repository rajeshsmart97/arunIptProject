Steps to Git Push :

Step 1 : For Configuaration
 
    git config --global user.email "emailid"
    git config --global user.name "username"

step 2: Initialize the local Repository

     git init

step 3:  Add all the files into local repository
  
     git add .  (before . give space)
 ---> all files

Step 4: For Declaration - Staging Area
 
     git commit -m "To Pass message what we want"

step 5: Connect to Remote Repository

     git remote add origin repositoryUrl

step 6: Push to remote master

     git push -u origin master
